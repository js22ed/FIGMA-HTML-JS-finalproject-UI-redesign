# User Interface Desktop Website Design Project

## Overview

This project is a Website User Interface design project that uses Figma, React JS, HTML to create a functioning simplistic planetary website suitable for all ages. 

## Problem

Many website informational pages are:

* Bland
* Confusing navigation
* Too many steps

## Solution

I designed the flow to:

* Be simple - I went with a very minimalistic, simple design because I believe 
having a simple mental model (law of similarity) helps with younger and older generations 
ability to learn and understand the website design easier.

* Be connected -  The law of uniform connectivity was a concept that I followed pretty closely
as all the small designs (circles for planets) are linked to represent the connection between
them and allows the users focus to remain either on the information, or the graphic. It helps
keep everything simple, colourful, and connected.

* Be uniform - The bursts of colour was very important for me as I decided to maintain a 
extensive colour pallette without much restriction. It flows in a nice way and nothing 
stands out too much that will draw the users away, I believe the bursts of colour helps with 
engagement and interactivity, as not all planets are the colour, it helps younger and older 
generation differentiate the different planets in a easy way. 

## Tools Used

* Figma
* HTML
* React Java Script


## Design Process

1. Analyzed existing informational desktop website applications
2. Identified exaustion points
3. Analysed wireframes through Wirify
4. Designed Wireframes
5. Designed interaction points
6. Coded with HTML / React JS
7. Built style guide

## Key Features

* Clean, creative UI
* Clear interaction buttons
* Improved user flow
* Consistent design
